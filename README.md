# Next.js App Router Study Archive

This project is a learning archive for studying **Next.js App Router** concepts and patterns.

Each commit represents a focused implementation or experiment with a specific App Router feature (routing, layouts, loading states, streaming UI, Server Actions, caching strategies, error handling, etc).  
By reviewing the commit history, you can easily revisit individual topics and understand how each feature was implemented step by step.

## Purpose

- Practice and document Next.js App Router behavior
- Build hands-on examples for core routing and rendering concepts
- Organize learning progress using commit-based history
- Create a long-term reference project for future development

## How to Use This Repository

- Browse commit history to review individual topics and feature implementations
- Checkout specific commits to study isolated examples
- Use this repository as a reference when building real-world App Router projects

## Covered Topics (Examples)

- File-based routing with App Router
- Layouts and nested routes
- Loading UI and Suspense streaming
- Server Components and Client Components
- Server Actions and form handling
- Data fetching and caching strategies
- Error handling and not-found pages
- Dynamic routes and route segments

## Getting Started

```bash
npm run dev
# or
yarn dev
```
